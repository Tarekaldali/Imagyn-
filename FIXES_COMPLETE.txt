## 🚀 HOW TO START THE APPLICATION:

### Option 1: Using START.bat (RECOMMENDED)
```batch
Double-click START.bat in the root folder
```
This will automatically start:
- ComfyUI Backend (Port 8189)
- Backend API Server (Port 8000)
- Flask Web Wrapper (Port 8890)

### Option 2: Manual Start
```powershell
# Terminal 1 - ComfyUI
.venv\Scripts\python.exe main.py --port 8189

# Terminal 2 - Backend API
cd ai_platform\backend
venv\Scripts\python.exe -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# Terminal 3 - Flask Wrapper
cd web_wrapper
..\.venv\Scripts\python.exe flask_server.py
```

## 🔧 TROUBLESHOOTING:

### Issue: "fetch failed" on login/register
**Solution**:
```powershell
# 1. Ensure backend is running
cd ai_platform\backend
venv\Scripts\python.exe -m uvicorn app.main:app --port 8000

# 2. Check if port 8000 is accessible
curl http://localhost:8000/health

# 3. Check browser console for specific error
```

### Issue: Backend not starting
**Solution**:
```powershell
# Check .env file exists
Test-Path ai_platform\backend\.env

# Install/update dependencies
cd ai_platform\backend
venv\Scripts\pip install -r requirements.txt
```

### Issue: Images not blurring for guests
**Solution**:
- Clear browser cache (Ctrl+F5)
- Check browser console for JavaScript errors
- Ensure you're accessing via `http://localhost:8890`

### Issue: ComfyUI not generating images
**Solution**:
```powershell
# Check if ComfyUI is running
curl http://localhost:8189/queue

# Check models are loaded
curl http://localhost:8189/object_info
```

## 🔐 SECURITY NOTES:

⚠️ **IMPORTANT**: Current CORS setting `allow_origins=["*"]` is for DEVELOPMENT ONLY!

Before deploying to production, update `ai_platform/backend/app/main.py`:
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=[
        "https://your-production-domain.com",
        "https://www.your-production-domain.com"
    ],
    allow_credentials=True,
    allow_methods=["GET", "POST", "PUT", "DELETE", "OPTIONS"],
    allow_headers=["*"]
)
```
